### Switch case
```
switch (variable) {
    case value1:
        // Code
    case value2:
        // Code
    default:
        // Code
}
```

### Loop
for loop
```
for (int i = 1; i<10; i++) {
    // Code
}
```

while loop
```
while (int i<10) {
    i++;
    // Code
}
```

do while loop
```
do {
    i++
    // Code
} while (int i < 10)
```
